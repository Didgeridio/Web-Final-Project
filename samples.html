Project Samples

I designed a multiplatform mobile application using MAUI.
The application uses the API of a popular MMO to allow the user to track marketboard prices and get alerts when items drop below their tracked price.
It also allows the user to load and save profiles, as well as change the theme from light to dark and set the search language from ENG to JP.
It is coded using MVVM principles following the official Microsoft guidelines.

I designed a Python text based RPG designed to be played on terminal.
It follows the player as they navigate their office, trying not to get fired after forgetting the Sales Report for their boss.
The player can interact with people and objects and there are various conditionals throughout the story.
It is coded using OOP principles.
